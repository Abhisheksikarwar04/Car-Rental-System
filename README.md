Car Rental System 🚗

A console-based Car Rental Management System developed in Java using Object-Oriented Programming (OOP) principles.
This project allows users to rent and return cars, manage customer records, track rental details, and calculate rental costs dynamically.

📌 Features
* View available cars
* Rent a car
* Return a rented car
* Customer management
* Rental tracking system
* Dynamic rental price calculation
* Menu-driven console interface
* Object-Oriented design implementation

🛠️ Technologies Used
* Java
* OOP Concepts
    * Classes & Objects
    * Encapsulation
    * Abstraction
    * Composition
* Java Collections Framework
* Scanner Class for user input

⚙️ Functional Modules

🚘 Car Management
* Store car details
* Check availability status
* Calculate rental cost

👤 Customer Management
* Add customer records
* Generate customer IDs

📋 Rental Management
* Create rental transactions
* Track rented cars
* Return car functionality

🧠 OOP Concepts Used

| Concept       | Usage                          |
| ------------- | ------------------------------ |
| Encapsulation | Private fields with getters    |
| Abstraction   | Separate classes for entities  |
| Composition   | Rental contains Car & Customer |
| Modularity    | Organized class structure      |


▶️ How to Run
1. Clone Repository

   git clone https://github.com/Abhisheksikarwar04/car-rental-system.git

2. Navigate to Project Folder

   cd car-rental-system

3. Compile Java File

   javac Main.java

4. Run Application

   java Main
